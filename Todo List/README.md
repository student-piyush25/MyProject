
# To-Do List Application

## 📌 Description
A simple **Python-based To-Do List** application that allows users to manage their daily tasks with file storage.

## ✨ Features
✅ Add tasks  
✅ Delete tasks  
✅ View saved tasks  

## 🛠️ Technologies Used
- Python
- File Handling

## 🚀 How to Run
1. Open a terminal or command prompt.
2. Run the script:
   ```sh
   python main.py
   ```
